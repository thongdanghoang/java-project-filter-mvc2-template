Các tập tin làm view là html và jsp
- Khi nào trang web đông có xử lý dữ liệu động mới làm trang jsp
- Các trang nhập liệu, chứa form thì dùng html
- Việc bắt lỗi nhập liệu thì dùng JS ở trang html
Việc mapping gọi action, resource và liên kết giữa các trang phải là các label 
 được mapping trong file properties để thông qua attribute của context scope để
 tham chiếu và lấy ra.
- Tuyệt đối không được đưa đường dẫn url cụ thể của bất kỳ resource nào
